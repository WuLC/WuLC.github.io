е‘Ҫд»ӨиЎҢзј–иҜ‘Javaжәҗж–Үд»¶

еҸ‘иЎЁдәҺ 2016-01-13 ж Үзӯҫ Java

дј—жүҖе‘ЁзҹҘпјҢJavaжҳҜдёҖй—Ёзј–иҜ‘еһӢиҜӯиЁҖпјҢйңҖиҰҒзј–иҜ‘жҲҗеӯ—иҠӮз ҒжүҚиғҪеңЁJVMдёҠиҝҗиЎҢгҖӮеёёз”Ёзҡ„IDEеҰӮEclipseзӯүе°Ҷзј–иҜ‘гҖҒиҝҗиЎҢзӯүжӯҘйӘӨз»“еҗҲиө·жқҘдёҖиө·жү§иЎҢпјҢеҸӘйңҖиҰҒжҢүдёӢRunеҚіеҸҜе®ҢжҲҗзј–иҜ‘е’ҢиҝҗиЎҢзҡ„е·ҘдҪңгҖӮдҪҶжҳҜе®һйҷ…дёҠзј–иҜ‘javaзЁӢеәҸзҡ„ж ёеҝғжҳҜJDKгҖӮжң¬ж–Үдё»иҰҒи®Іиҝ°дәҶеҸӘе®үиЈ…jdkйҖҡиҝҮе‘Ҫд»ӨиЎҢжқҘзј–иҜ‘иҝҗиЎҢJavaзЁӢеәҸгҖӮ

JDK дёҺ JRE

иҜҙеҲ°JDKе’ҢJREзҡ„е…ізі»гҖӮеҸҜд»Ҙз®ҖеҚ•и®ӨдёәJDKжҳҜз”ЁжқҘзј–иҜ‘javaж–Үд»¶зҡ„пјҢиҖҢjreжҳҜз”ЁжқҘиҝҗиЎҢзј–иҜ‘з”ҹжҲҗзҡ„classж–Үд»¶гҖӮJDKдёӯеҢ…еҗ«JREпјҢеңЁJDKзҡ„е®үиЈ…зӣ®еҪ•дёӢжңүдёҖдёӘеҗҚдёәjreзҡ„зӣ®еҪ•пјҢйҮҢйқўжңүдёӨдёӘж–Үд»¶еӨ№binе’ҢlibпјҢеңЁиҝҷйҮҢеҸҜд»Ҙи®ӨдёәbinйҮҢзҡ„е°ұжҳҜjvmпјҢlibдёӯеҲҷжҳҜjvmе·ҘдҪңжүҖйңҖиҰҒзҡ„зұ»еә“пјҢиҖҢjvmе’Ң libе’Ңиө·жқҘе°ұз§°дёәjreгҖӮ

еҸҰеӨ–еңЁwindowsдёӢе®үиЈ…jdkжҳҜдјҡжңүйҖүжӢ©жҳҜеҗҰе®үиЈ…еҚ•зӢ¬зҡ„jreпјҢеҰӮжһңйҖүжӢ©е®үиЈ…пјҢйӮЈд№Ҳе°ұдјҡе®үиЈ…дәҶдёӨеҘ—jreпјҢдёҖеҘ—еҢ…еҗ«еңЁjdkе®үиЈ…зӣ®еҪ•дёӯпјҢеҸҰеӨ–дёҖеҘ—еҲҷжҳҜзӢ¬з«Ӣе®үиЈ…зҡ„jreгҖӮйӮЈд№ҲиҝҷдёӨеҘ—jreжңүеҢәеҲ«еҗ—пјҹ

зӯ”жЎҲжҳҜжңүзҡ„пјҢеӣ дёәjdkжҳҜдҪңдёәејҖеҸ‘дҪҝз”Ёзҡ„пјҢиҖҢjreд»…д»…жҳҜдҪңдёәиҝҗиЎҢjavaзЁӢеәҸзҡ„зҺҜеўғгҖӮеӣ жӯӨе…·еӨҮејҖеҸ‘еҠҹиғҪзҡ„jdkиҮӘе·ұзҡ„jreдёӢдјҡеҗҢж—¶жңүclientжҖ§иҙЁзҡ„jvmе’ҢserverжҖ§иҙЁзҡ„ jvmпјҢиҖҢд»…д»…дҪңдёәиҝҗиЎҢзҺҜеўғзҡ„jreдёӢеҸӘйңҖиҰҒclientжҖ§иҙЁзҡ„jvm.dllе°ұеӨҹдәҶгҖӮ

еҸҜж №жҚ®йңҖиҰҒзј–иҜ‘зҡ„жәҗж–Үд»¶зҡ„ж•°зӣ®е’ҢдҪҚзҪ®е°ҶйҖҡиҝҮе‘Ҫд»ӨиЎҢзј–иҜ‘иҝҗиЎҢjavaзЁӢеәҸеҲҶжҲҗдёӨеӨ§зұ»гҖӮдёҖзұ»жҳҜеҸӘжңүеҚ•дёҖзҡ„жәҗж–Үд»¶пјҢеҸҰеӨ–дёҖз§ҚжҳҜжңүеӨҡдёӘжәҗж–Үд»¶гҖӮ
